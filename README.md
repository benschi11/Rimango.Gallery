Rimango.Gallery
===============

An simple Gallery Module for Orchard CMS.

Demo coming soon...

Version 0.1 (Alpha)
------------------
* Adds 2 new ContentTypes: Gallery and GalleryEntry
* Allows to create multiple Galleries with multiple Gallery Entries
* Default Views for Gallery - easy to override
